# Tesao MagicMirror
A digital signage tool to create beautiful life by magicmirror.

## Directory architecture
The entity of MagicMirror is running on raspberry pi 5.
So this directory manages only settings and custom modules.

- **config.js**: Configuration of MagicMirror.
- **modules/**: Custom modules created by user.

## Deploy
Modules are deployed with `scp` command run by user.
You don't need to think about deploy.
